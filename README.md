# Amazon Entity Resolution (amazon-entity-resolution)
Amazon Entity Resolution is a service that helps you match and link related records across multiple applications, channels, and data stores using machine learning and configurable matching techniques to identify and consolidate records that refer to the same entity.

**URL:** [https://aws.amazon.com/entity-resolution/](https://aws.amazon.com/entity-resolution/)

## Features

| Name | Description |
|------|-------------|
| ML-Based Matching | Use machine learning models to match records across disparate datasets |
| Rule-Based Matching | Configure deterministic matching rules for exact and fuzzy matching |
| ID Mapping | Create and manage identity graphs linking records across data sources |
| Schema Mapping | Map input data schemas to standardized formats for consistent matching |
| Third-Party Data Providers | Enrich records with data from LiveRamp, Unified ID 2.0, and others |

## Use Cases
