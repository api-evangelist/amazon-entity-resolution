# Amazon Entity Resolution (amazon-entity-resolution)
Amazon Entity Resolution is a service that helps you match and link related records across multiple applications, channels, and data stores. It uses machine learning and configurable matching techniques to identify and consolidate records that refer to the same entity, such as a customer or product.

### Amazon Entity Resolution API
The Amazon Entity Resolution API provides programmatic access to create and manage matching workflows, schema mappings, and ID mapping tables for matching and linking related records across data sources.
